# Resolução de Problemas com Grafos

Este repositório reúne **todos os códigos utilizados na disciplina de
Resolução de Problemas com Grafos**.

Aqui estão disponíveis **implementações de algoritmos**, **exemplos utilizados
no material didático** e **códigos relacionados às listas de exercícios**,
organizados de forma clara para consulta e estudo.

---

##  Objetivo

Centralizar, em um único repositório, todo o código-fonte utilizado na disciplina,
permitindo que o aluno:

- Estude os algoritmos de grafos apresentados em sala e na apostila  
- Analise implementações práticas dos principais métodos  
- Utilize os códigos como apoio para resolução de exercícios e projetos  

Não há separação por aulas.  
**Todo algoritmo ou exemplo citado no material didático está disponível neste repositório.**

---
##  Conteúdos Abordados

Entre os principais algoritmos disponíveis neste repositório, destacam-se:

- Busca em Largura (BFS)
- Busca em Profundidade (DFS)
- Algoritmo de Dijkstra
- Algoritmo de Bellman-Ford
- Algoritmo de Floyd-Warshall

---

##  Público-Alvo

- Alunos de graduação em Computação e áreas correlatas  
- Estudantes cursando disciplinas de Grafos e Estruturas de Dados  
- Professores que utilizem grafos como ferramenta de resolução de problemas  

---

##  Observações Importantes

- Os códigos têm **finalidade exclusivamente educacional**
- Algumas implementações priorizam clareza didática em vez de otimização extrema
- Recomenda-se o estudo conjunto com o material teórico da disciplina
